» UPGRADE:
Upgrading is only for the versions shown below. Please follow the instructions carefully. To see whats been added in each version, please see the following file:

docs/history.txt

Note that not all versions are guaranteed to be upgradeable from the last.

» UPGRADE (v3.1 => v4.0 ONLY):
Due to the changes made in v4.0, there are no upgrades available for this option. Please perform a clean install.
» UPGRADE (v3.0 => v3.1 ONLY):
There are no database changes with v3.1 By uploading the new templates you will delete any changes you made to your current system. To upgrade to v3.1 replace the following files on your server. Where a folder only is specified, you should replace the whole folder:

/admin/ (EXCEPT inc/password.inc.php) (Updated)
/classes/ (Updated)
/css/ (Updated)
/docs/ (Updated)
/images/ (Updated)
/inc/ (EXCEPT inc/db_connection.inc.php) (Updated)
/lang/ (Updated)
/templates/*tpl.php (ALL EXCEPT /email/ folder) (Updated)
index.php (Updated)
