» SCRIPT INFO:
Information about this script is shown below:
» FEATURES:
• Simple blog system for your website.
• Easily add/edit or delete blogs.
• Allow visitor comments for individual blogs.
• Optional e-mail notification for webmaster if comments are posted.
• Edit or delete visitor comments.
• Option to auto parse links in comments.
• Optional SMTP send mail options.
• Search engines to search blogs.
• Option for visitors to contact blog owner.
• Add profile and optional profile image.
• Time offset feature to adjust time display.
• XHTML/CSS theme based layout.
• Archiving system.
• Optional search engine friendly urls via .htaccess.
• Option to add adsense or snap code.
• Optional WYSIWYG editor for text emphasis.
• Upload/manage images and flash files via WYSIWYG editor.
• Easily embed YouTube videos in your blogs via WYSIWYG editor.
• Admin can use HTML in blogs or comments.
• Supports 16 social bookmark sites including Digg & Del.icio.us.
• RSS feed.
• Option to add favourite websites.
• Optional captcha code for forms.
• Printer friendly version of blogs/comments.
