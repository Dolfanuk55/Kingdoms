List of template files:
=======================

Below is a list of the system template files. For any theme, these files MUST be in place. Do not rename
any of the files once they have been created. The system checks these files are in place and displays an error
if any are missing if the 'CHECK_TEMPLATES' var is enabled. See the docs for more info.

If you don`t need a file, create a blank file with the same name. This will act as a place holder.
